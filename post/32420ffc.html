添加Python解释器

  1. 使用指令 which python 获取系统中python的安装路径
  2. 打开 配置-偏好设置
  3. 添加Python环境，点击红框圈中的设置按钮
  4. 选择系统中的环境，并找到python的路径，选择该路径下的python3文件即可
  5. 选择成功后就出现以下界面。点击Apply应用即可
